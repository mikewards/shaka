Contributing
============

Pull requests welcome.

Before submitting:

1. Run tests: `./gradlew test` (API), `flutter test` (app)
2. Format code: `./gradlew ktlintFormat` (API), `dart format .` (app)
3. Keep commits atomic and descriptive

Commit authorship
-----------------

All commits must be authored and committed as **mikewards**
(`zmikewardz@gmail.com`). Do **not** commit as, or attribute commits to,
`cursoragent` / "Cursor" in any field:

- No `cursoragent` (or other bot) in the author or committer fields.
- No `Co-authored-by: Cursor <cursoragent@cursor.com>` trailer in commit messages.

If using Cursor, disable the "Co-authored-by Cursor" git setting so the trailer is
not auto-added.

Report bugs and feature requests via GitHub Issues.
